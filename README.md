# GAK-Firnasari
Repository ini dibuat untuk memenuhi tugas mata kuliah Grafika Komputer.
Proyek ini merupakan game sederhana **Collect Money Game** yang dibuat menggunakan bahasa pemrograman Python.

# Cara Menjalankan Proyek
1. Pastikan Python sudah terinstal di komputer.
2. Download repository ini atau lakukan clone repository dari GitHub ke komputer.
3. Pastikan semua file berikut berada dalam satu folder:
   - praktikum10_Collect-Money-Game_23067.py
   - bom.png
   - latar.jpeg
   - orang.png
   - uang.png
4. Buka folder proyek menggunakan Command Prompt / Terminal.
5. Jalankan program dengan perintah berikut:
   - python praktikum10_Collect-Money-Game_23067.py
6. Setelah perintah dijalankan, program game akan terbuka dan dapat dimainkan sesuai dengan aturan yang telah dibuat.
